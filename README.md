# ThreatForge AI

ThreatForge AI is an AI-powered cybersecurity intelligence platform designed to help security professionals analyze threats, monitor security events, and gain actionable insights through an interactive dashboard and AI copilot.

## Features

### AI Cybersecurity Copilot

* Interactive security assistant
* Threat analysis support
* Phishing awareness guidance
* Malware investigation assistance
* Password security recommendations

### Security Dashboard

* Real-time threat monitoring
* Threat trend visualization
* Live security activity feed
* AI-generated threat analysis
* Security operations overview

### Threat Intelligence

* Threat score monitoring
* Active threat tracking
* Protected asset visibility
* Incident monitoring
* Security analytics

## Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/threatforge-ai.git
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd threatforge-ai
npm install
npm run dev
```

## Environment Variables

Create a `.env` file inside the backend directory.

```env
MONGODB_URI=YOUR_MONGODB_CONNECTION_STRING
PORT=5000
```
